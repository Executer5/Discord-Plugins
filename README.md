# 🚀 Discord File Sharing Plugin

A lightweight Discord plugin that extends Discord's file-sharing capabilities by automatically splitting large files into smaller parts during upload and rebuilding them on supported clients.

The plugin is designed to make sharing large files simple and reliable while keeping the experience fully integrated within Discord. Users can upload files using the normal Discord interface, while the plugin handles chunking, transfer management, and reconstruction automatically in the background.

## ✨ Features

* Automatic file splitting for large uploads
* Seamless file reconstruction on supported clients
* Support for images, videos, archives, documents, and other file formats
* Upload progress tracking
* Inline image previews
* Local file reconstruction
* Lightweight and easy to use
* No external file hosting services required

## 🔒 Security

Files are transferred through Discord's existing attachment system. No third-party storage providers or external servers are used. File reconstruction takes place locally on the receiving client.

## 💻 Compatibility

Compatible with supported Discord mod clients and designed to work alongside existing Discord workflows without requiring additional file-sharing platforms.
